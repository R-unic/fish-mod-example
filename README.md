# Fish Mod Example

This is a repo for an example mod for my game Neanderthals. It adds Raw Fish and Cooked Fish.

To add assets to your mod, you must first upload them onto Roblox and give Neanderthals permission to use your asset.

Here's how:

- Configure your asset
- Go to the "Permissions" tab
- Add the Neanderthals experience (ID 8118938974).
